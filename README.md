# Summer Research 2026: Computational Agronomy

This repository contains the code, data, and research materials for the **Computational Agronomy Summer Intensive (2026)**. The project focuses on bridging Plant Breeding and Genetics (PBG) with software engineering and Artificial Intelligence to improve field trial phenotyping and localized genomic/phenotypic analysis.

## Objectives
- **Offline-First Phenotyping Tool**: Develop a high-utility, offline-first phenotyping application for field trials.
- **Edge-AI & LLMs**: Leverage edge-based artificial intelligence and large language models for localized genomic and phenotypic analysis.
- **Research Manuscript**: Draft a publication-ready manuscript detailing the methods, tools, and findings.

## Repository Structure
- `research_env/`: Python virtual environment containing the required dependencies.
- `src/`: Source code for the phenotyping tool and data processing scripts.
- `notebooks/`: Jupyter notebooks for data analysis and modeling.
- `data/`: Local data directories (git-ignored where appropriate).
- `docs/`: Research documentation, protocols, and draft manuscript sections.

## Getting Started
### Prerequisites
- Python 3.10+
- Git

### Setup
1. Clone the repository.
2. Activate the virtual environment:
   - **Windows (PowerShell)**:
     ```powershell
     .\research_env\Scripts\Activate.ps1
     ```
   - **Unix/macOS**:
     ```bash
     source research_env/bin/activate
     ```
3. Run the analysis or start the application.
